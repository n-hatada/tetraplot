 概要
======
TetraPlotは四元系の化学組成をプロットするためのプログラムです。

 動作環境
==========
Windows XPまたはそれ以上
(動作確認はWindows XP及びWindows 7、Windows 10で行いました。)

 インストール
==============
"TetraPlot1.x.x_Setup.exe"を実行し指示に従ってください。

 アンインストール
==================
コントロールパネルでTetraPlotをアンインストールしてください。

 簡単な使い方
==============
1. スタートメニューの中の「TetraPlot」を起動します。
2. 「軸の名前」部に軸の名前（成分名）を入力します。
3. 「系列一覧」部で「追加」をクリックして系列を追加した後、それを選択します。
4. 系列の情報を「選択された系列の詳細」部で編集します。
5. 「Diagram」ウィンドウに四面体形の組成図が表示されます。 

添付のtpxmlファイルを開き、四面体形の組成図の例をご覧ください。

 ベクター形式の画像の保存
==========================
現在TetraPlotにはベクター形式の画像の保存機能はありません。しかし、*.emf、*.eps、*.svg等の形式のファイルが保存できる仮想プリンターに図を印刷することにより、ベクター形式の画像を得ることが可能です。
